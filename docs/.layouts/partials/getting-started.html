Getting started with Sourcegraph


Sourcegraph is a Git repository service with Code Intelligence. It runs on your own server or cloud and installs in 5 minutes. You may also use Sourcegraph on Sourcegraph.com for free.

Use Sourcegraph.com

Create an account on our cloud.

Install on your own server

Sourcegraph requires Docker and PostgreSQL. Install those first, then continue with the instructions below.

Note: our install script for the cloud providers listed below will automatically install Docker and PostgreSQL.

Mac OS X

curl -O -L https://sourcegraph.com/.download/latest/darwin-amd64/src.gz
gunzip src.gz
chmod +x src
./src serve

Linux

curl -O -L https://sourcegraph.com/.download/latest/linux-amd64/src.gz
gunzip src.gz
chmod +x src
./src serve
